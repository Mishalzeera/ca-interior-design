Lato Interior Design


Theatres need to be at something like 70% capacity to break even. As an actor looking out as an auditorium fills up, if it’s only half full you go: “Oh bloody hell, bad house tonight.” So the idea that we can make plays work when you’ve only got half a house is not really practical. If you’ve got a two-hander you might be able to get by. But if you’re doing something like Les Misérables, you’ve got a massive cast, costumes, all the stage hands, an orchestra. You can’t pay for it.
